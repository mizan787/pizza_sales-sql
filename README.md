# Pizza Sales SQL Project

## Description
Sales analysis project using Microsoft SQL Server.

## Tables
- orders
- order_details
- pizzas
- customers

## Features
- Normalized schema
- Foreign key relationships
- Bulk insert support
- Sales analytics queries

## How to run
1. Create database in SQL Server
2. Run pizza_sales.sql
3. Make sure CSV files are in same folder

## Tools Used
- Microsoft SQL Server
- SSMS
